General Utilities¶

This module provides a collection of miscellaneous utilities to facilitate data analysis, saving plots, OS manipulations and etc.

mkdir¶

utilities.mkdir(directory)¶

This function check if the provided directory exists and if not it creates a new one :param directory: directory to be checked and created. :type directory: str

Example

>>> PythonTools.mkdir('E:/Git_Repo')

plot setting¶

utilities.plot_settings(style=None, font_family='Arial')¶

This function check applies EDW plot settings to matplotlib tables. This may be used in jupyter notebooks.

Parameters:
  • style (str) – Matplotlib permissible styles
  • font_family (str) – The desired font family

Example

>>> PythonTools.plot_settings(style = 'classic', font_family = 'Arial')

Random RGB Color Generator¶

utilities.random_rgb_color(ncolors=20, seed=69069)¶

This function generates an array of colors using numpy random number generator. The seed number can be used to control the random number generator sequence.

Parameters:
  • ncolors (int) – Desired number of colors
  • seed (int) – Random number generator seed number

Example

>>> RandomColor = PythonTools.random_rgb_color(ncolors = 20, seed = 69069)

Save Plot¶

utilities.save_plot(outfl=None, dpi=300, pad_inches=0.02, **kwargs)¶

This function can be used to save plots in different formats (i.e., png, jpg, pdf,…)

Parameters:
  • outfl (str) – Output address (directory + file name)
  • dpi (int) – dpi for the saved plot
  • pad_inches (float) – Pad size around the plot
  • **kwargs – Optional permissible keyword

Example

>>> RandomColor = PythonTools.save_plot(outfl='E:/Git_Repo/Figure1.pdf', dpi=300, pad_inches =0.02)

Unzip file¶

utilities.unzip_file(path_to_zip_file=None, destination=None)¶

This function can be used to unzip a zip file

Parameters:
  • path_to_zip_file (str) – Address of the zip file
  • destination (str) – destination for unzipped file

Example

>>> PythonTools.unzip_file(path_to_zip_file = 'Well_Reports.zip', destination = Well_Reports )

Get File Size¶

utilities.get_file_size(file_path=None, KB=1024, scale=None, size_in_bytes_only=False)¶

This function can be used to get a file path an return the file size

Parameters:
  • file_path (str) – Path/address for the target file
  • KB (int) – Measure of every kilo bytes (1024/1000)
  • scale (str) – Desired scale (‘bytes’, ‘KB’, ‘MB’, ‘GB’, ‘TB’). ‘TB’ is used of scale is None

Example

>>> PythonTools.get_file_size(file_path = 'Well_Reports.mdb',KB=1000, scale='MB')

Monthly Increment for a Date¶

utilities.add_month(input_date=None, num_months=1)¶

This function can be used to increment datatime.date by specified number of months

Parameters:
  • input_date (datatime.date) – input date
  • num_months (int) – number of months for each increment

Example

>>> PythonTools.add_month(input_date = datetime.date.today(), num_months = 1))
