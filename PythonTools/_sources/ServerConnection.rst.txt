Connection to a Database
##########################

PythonTools serves as a data analysis/mining tool. It is often required to import data from a database to perform transformation, analysis and visualization. PythonTools has a connection module that allows the user to connect to different servers. The result of the query which is a tables is casted into pandas data frames.
