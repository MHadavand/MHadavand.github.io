Connection to a Database¶

PythonTools serves as a data analysis/mining tool. It is often required to import data from a database to perform transformation, analysis and visualization. PythonTools has a connection module that allows the user to connect to different servers. The result of the query which is a tables is casted into pandas data frames.

Class: ServerConnect¶

class PythonTools.ServerConnect(server=None, database=None, trusted_connection=True, uid=None, pwd=None)[source]¶

This is a class to facilitate connection to SQL server databases using Python implementation of ODBC (pyodbc).

Parameters:
  • server (str) – Server name/address for connection
  • database (str) – Database name/initial catalog
  • trusted_connection (bool) – True if Windows Authentication is used
  • uid (str) – User ID to login and access the server
  • pwd (str) – Password

Example

Quick connection into local server:

>>> connection=PythonTools.ServerConnect(server='localhost',database='EdwInfoMartLocal', trusted_connection=True)

Code author: Mostafa Hadavand 2017-26-7
